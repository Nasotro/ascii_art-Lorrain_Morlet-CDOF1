# ASCII Lorrain MORLET

This project is a very simple implementation of an ASCII conversion of the monalisa to a text file with ascii characters. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

You need python installed on your machine.

### Installing

A step by step series of examples that tell you how to get a development env running. 

First of all you need to clone this repo on your machine.
You can navigate to the folder where you want to clone the repo by using this command:
```
cd path/to/your/folder
```

Then you can clone the repo with the following command:
```
git clone https://github.com/Nasotro/ascii_art-Lorrain_Morlet-CDOF1.git
```

You also need the following python librairies installed on your machine.
- numpy
- openCV
You can you use the following commands to install them:
```
pip install numpy
pip install opencv-python
```

## Running the Code

In order to run the code you need to navigate to the folder where you cloned the repo and run the following command:
```
python images_ascii.py
```

4. **Make your changes and commit them:**
